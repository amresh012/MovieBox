
# Redux Movie APP with React-Redux 




## Demo

https://movie-box-puce-seven.vercel.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/amresh012/MovieBox.git
```

Go to the project directory

```bash
  cd MovieBox
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
